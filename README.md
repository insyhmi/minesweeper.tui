# minesweeper.tui
Terminal Minesweeper

# Controls

| Control | Action |
| :------- | :------: |
| Arrow Keys | Move Cursor |
| Space | Flag Cell |
| Enter | Reveal Cell |

# Run
## 1. .exe file
The game will run with the default parameter of 10x10 board with 15 mines
## 2. Command Line
In the working directory, ```./minesweeper [height] [width] [mine_count]```
